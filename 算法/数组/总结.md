
## 1: 二分查找

### 1:使用的场景

有序，非重复的数组可以考虑使用二分查找。

### 二分查找的套路

``` javaScript
const search = function (nums, target) {
  let start = 0;
  let end = nums.length - 1;

 // 注意这里是小于等于。 
  while (start <= end) {
    let mid = Math.floor((start + end) / 2);
    let m = nums[mid];

    if (m === target) {
      return mid;
    }
    if (m < target) {
      start = mid + 1;
    }
    if (m > target) {
      end = mid - 1;
    }
  }

  return -1;
}
```
